# Weather Data Prediction

## Overview
A Decision Tree Classifier Model created to predict the weather conditions of Delhi.

## Prerequisites
* Python 3 
* an IDE (preferably Jupyter Notebook)
* Pandas 
* Numpy 
* Sklearn
* Seaborn 
* Graphviz
* Matplotlib
